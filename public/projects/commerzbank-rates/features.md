## Authentication & Authorisation 
- Users can sign in using email/password
- Users can sign in using Microsoft Entra 
- Users can log out
- Users can view their profiles 
- Admins can manage applications based on Entra roles 

## Viewing Rates 
- Users can view risk free rates for the week
- Users can view forward looking forms rates for the year
- Users can view rates for any date, currency and type
- Users can export rates into spreadsheets based on date, currency and type

## Administrator Functionality
- Admins can set and enable/disable polling schedules for each currecy
- Admins can manually fetch currencies for the past week (in case of missing data or automatic poll failure)
- Admins can modify retry for automatic polling based on type and duration
- Admins can tweak data validation configurations such as stale data checks and time tollerance 
- Admins can set webhook alerts (for success or failures) which are sent to specific rooms or people using MessageML
- Admins can set email alerts (for success or failures)
- Admins can set app messages (such as banners, toasts and dialogs) that are shown when users navigate to the app
- Admins can change contact details 
- Admins can view rate statistics 
- Admins can view available currencies 
- Admins can view versions of various tools such as Java, Spring Boot and app version
